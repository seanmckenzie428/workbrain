# WorkBrain

A native macOS daily notes application with always-on-top pinning and adjustable transparency.

## Features

- **Daily Notes** - One markdown note per day, auto-created with customizable template
- **Day Selector** - Calendar picker to view and edit any day's notes
- **Always-on-Top** - Pin window above all other apps
- **Transparency Control** - Adjust window opacity in real-time
- **Liquid Glass Design** - Modern frosted glass aesthetic

## Requirements

- macOS 14.0 (Sonoma) or later
- Xcode 15.0 or later
- Swift 5.9+
